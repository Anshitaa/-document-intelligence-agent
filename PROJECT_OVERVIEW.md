## 🚀 Getting Started

1. **Clone and setup**:
   ```bash
   git clone <your-repo>
   cd document-intelligence-agent
   pip install -r requirements.txt
   ```

2. **Add your API key**:
   ```bash
   echo "OPENAI_API_KEY=your_key" > .env
   ```

3. **Add PDF documents**:
   ```bash
   mkdir data
   cp your_documents.pdf data/
   ```

4. **Run the agent**:
   ```bash
   python document_intelligence_agent.py
   ```
